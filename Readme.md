# Job Market Intelligence Dashboard (Power BI)

## 1. Project Overview
This project analyzes job posting trends using a LinkedIn job dataset to uncover in-demand roles, 
hiring patterns and emerging job opportunities.
The goal was to build a decision-support dashboard that helps:
  - Job seekers identify high-demand roles
  - Analysts understand hiring trends
  - Recruiters track talent demand

## 2. Dataset Description
The dataset consists of these columns:
  1. job_title:	Name of the role
  2. company_name:	Hiring company
  3. city:	Job location city
  4. state:	Job location State
  5. country:	Job location Country
  6. hiring_status:	Actively Hiring
  7. date:	Posting date
  8. seniority_level: Intern, Associate, Entry, Mid-Senior, Senior, Director
  9. job_function:	Engineering, Marketing, Sales, etc
  10. employment_type:	Full-time, Contract, Internship, Volunteer, Temporary, Part-Time
  11. industry
